## Hướng dẫn cài đặt sơ bộ dành cho nhóm
1. Kéo (Clone/Pull) mã nguồn từ repository này về, đặt vào thư mục `htdocs` của XAMPP (ví dụ: `C:\xampp\htdocs\datvexemphim`).
2. Mở bảng điều khiển XAMPP, nhấn **Start** cho 2 module là **Apache** và **MySQL**.
3. Tại dòng MySQL trên XAMPP, nhấn nút **Admin** để trình duyệt tự động mở giao diện quản trị cơ sở dữ liệu (phpMyAdmin).
